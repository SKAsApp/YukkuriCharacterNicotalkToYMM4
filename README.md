# キャラ素材をnicotalk形式からゆっくりムービーメーカー4形式に変換するヤツ ／ YukkuriCharacterNicotalkToYMM4

<!-- convert nicotalk’s yukkuri character material to Yukkuri Movie Maker 4’s character -->
